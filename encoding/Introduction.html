Welcome to the Encoding Task Experiment

Thank you for participating in our study. This experiment is designed to investigate the process of encoding in memory, focusing on a simple letter-to-number encoding task.

Experiment Overview

  • You will be presented with two letters: A and B
  • A corresponds to the number 1, and B corresponds to the number 2
  • The experiment consists of two rounds
  • In each round, you will be shown only one of the two letters, selected at random
  • Your task is to enter the correct numerical code for the letter shown
  • The letter and the input field for the code will be presented on the same page

Reward System

You will be awarded Â£5 for correctly encoding the letter. While there are two rounds, payment will be based on your performance in only one randomly selected round.

Instructions

  1. Pay close attention to the letter displayed on each round
  2. Enter the corresponding number (1 for A, 2 for B) in the provided input field
  3. Submit your answer
  4. Repeat for the second round

Your participation is valuable in helping us understand the cognitive processes involved in basic encoding tasks. If you have any questions, please ask the experimenter before beginning.
